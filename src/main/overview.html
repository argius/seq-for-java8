Seq-for-Java8 - the library of sequencial containers which is neither Collection nor Stream

Sequences in this library are simple array wrappers.

Features are:

  • immutable ArrayList (java.util.ArrayList)
  • some methods in Stream APIs.
  • primitive type supports (int, long, double)
  • Seq in Scala
All of these Sequences are not scalable. If you need scalability, use Streams in java.util.stream package.
